## 0.17

- **修正: アップデートのたびにライブラリ・既読状態・設定が数日前に巻き戻る問題**
  - 原因: ファイル単位のシンボリックリンクが、Suwayomi 本体の H2 マイグレーションや
    上流起動スクリプトの `sed -i` によって実ファイルに置き換えられ、以後のデータが
    コンテナ内(アップデートで消える領域)にしか書かれなくなっていた
  - 対策: Tachidesk データディレクトリ全体を `/config/tachidesk` に移して永続化。
    既存データは初回起動時に自動移行される
  - これによりダウンロード済みチャプターや各種設定もアップデートを跨いで保持される
- **重要(復旧手順)**: 前回アップデート以降のデータはコンテナ内にあったため、
  このアップデートの適用時点で失われる。適用「前」に Suwayomi の
  WebUI → 設定 → バックアップ で手動バックアップを作成しておき、
  適用「後」に WebUI から復元すると最新状態に戻せる
  (自動バックアップ(.tachibk)は `/addon_configs/<slug>/backups` に残っている)
